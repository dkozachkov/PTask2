# PTask2
Задание по практике 2: знакомство с языком Python
Описание задач:
1) сгенерировать случайное число в диапазоне от -1 до 1
2) сгенерировать n случайных чисел в этом же диапазоне
3) найти ошибки в заданном скрипте
4) печать логарифмов заданных чисел
5) запись данных в файл
6) чтение из файла
7) метод Монте-Карло: вероятность выпадения шестерки при бросании двух костей
8) игра в кости
